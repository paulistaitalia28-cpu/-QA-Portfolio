## Sobre este portfólio

Este repositório reúne os principais projetos que desenvolvi durante o **Bootcamp Analista de QA da TripleTen** (330h, janeiro a junho de 2026). Cada pasta representa uma sprint do curso e contém casos de teste, evidências de execução, bug reports e relatórios técnicos.

**O que você vai encontrar aqui:**

- Testes manuais estruturados (BVA, classes de equivalência, testes exploratórios)
- Testes cross-browser (Chrome, Firefox, Opera)
- Testes de API com Postman
- Automação Web com Selenium + Python + Pytest
- Bug reporting profissional no Jira e GitHub Issues
